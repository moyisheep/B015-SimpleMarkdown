## 📝 文本格式化示例

### 1. 标题级别
# 一级标题（H1）
## 二级标题（H2）
### 三级标题（H3）
#### 四级标题（H4）
##### 五级标题（H5）
###### 六级标题（H6）

### 2. 强调文本
这是**粗体文本**，这是*斜体文本*，这是***粗斜体组合***。
也可以使用下划线：__粗体__、_斜体_、___粗斜体___。

### 3. 删除线与标记
~~这是一段删除线文本~~
==这是一段高亮文本==（部分扩展支持）

## 📄 代码展示

### 行内代码
使用 `print("Hello, Markdown!")` 在段落中插入代码。

### 代码块（无高亮）
```
function example() {
console.log("这是一个代码块");
}
```

### 语法高亮
```javascript
// JavaScript 示例
const greeting = "Hello, Simple Markdown!";
console.log(greeting);

function calculate(a, b) {
return a + b * 2;
}
```

```python
# Python 示例
def hello_world():
print("Welcome to Simple Markdown!")
return True
```

## 📊 表格

| 功能 | 语法 | 示例 |
|------|------|------|
| 粗体 | `**文本**` | **示例** |
| 斜体 | `*文本*` | *示例* |
| 链接 | `[标题](URL)` | [点击这里](#) |
| 代码 | `` `代码` `` | `console.log()` |
| 左对齐 | 默认 | 文本 |
| 居中对齐 | `:---:` | 中间 |
| 右对齐 | `---:` | 右侧 |

**对齐控制：**
| 左对齐 | 居中对齐 | 右对齐 |
|:--------|:--------:|--------:|
| 单元格 | 单元格 | 单元格 |
| 较长内容示例 | 居中文本 | 右侧数字 |

## ✨ 提示与技巧

1. **混合使用**：你可以在列表中嵌套代码块，或在引用中使用表格。
2. **转义字符**：使用反斜杠显示特殊字符，如 \* 星号 \` 反引号。
3. **兼容性**：部分语法需要扩展支持（如 ==高亮==、脚注、公式等）。
